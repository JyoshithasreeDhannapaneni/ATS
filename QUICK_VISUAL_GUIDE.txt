QUICK ACTION PATHS
════════════════════════════════════════════════════════════════════════════

Path 1: VIEW QUESTIONNAIRE (Read-Only)
  Job Order List → Click Job Order → View Details → See Questionnaire

Path 2: CHANGE QUESTIONNAIRE
  Job Order Details → Click "Edit Questionnaire" → Select New One → Save

Path 3: ASSIGN QUESTIONNAIRE (if empty)
  Job Order Details → Click "Assign Questionnaire" → Select One → Save

KEY POINTS
════════════════════════════════════════════════════════════════════════════

Location:       Right column of job order details, after "Type" field
Always Shows:   For ALL job orders (public and private)
Display Format: Bold title if assigned, gray text if not assigned
Links:          "Edit Questionnaire" or "Assign Questionnaire"
Permissions:    Links only visible if you have edit permission
Integration:    Works with auto-selection feature perfectly
Career Portal:  Uses same questionnaire_id shown here
