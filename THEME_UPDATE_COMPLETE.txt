CUSTOMIZATION:
════════════════════════════════════════════════════════════════════════════

If you want to change colors:

1. Open: careersPage.css
2. Find: :root { section at the top
3. Change: --primary: #2563eb; to your color
4. Example: --primary: #10b981; (green instead of blue)
5. All elements automatically update!

No need to change individual colors - CSS variables handle it all!
