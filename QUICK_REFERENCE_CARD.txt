CHEAT SHEET - THE AUTO-SELECTION MAP:
════════════════════════════════════════════════════════════════════════════

When Creating a Job Order:

SELECT THIS TYPE → QUESTIONNAIRE AUTO-SELECTS TO:

[C]    Contract           ──→  Questionnaire ID 2
[C2H]  Contract to Hire   ──→  Questionnaire ID 3
[FL]   Freelance          ──→  Questionnaire ID 4
[FT]   Full Time          ──→  Questionnaire ID 5  ⭐ Most Common
[H]    Hire               ──→  Questionnaire ID 6
