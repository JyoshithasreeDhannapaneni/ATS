THREE SCENARIOS

SCENARIO 1: Job Order WITH Questionnaire Assigned

  Edit Questionnaire  (link)

  - Shows an "Edit Questionnaire" link
  - Click link to modify the questionnaire
  - Only visible if you have edit permission

SCENARIO 2: Job Order WITHOUT Questionnaire Assigned

When you open a job order with NO questionnaire:

  Questionnaire: No questionnaire assigned
  Assign Questionnaire  (link)

  Description:
  - Shows "No questionnaire assigned" in gray
  - Shows "Assign Questionnaire" link
  - Click link to go to Edit form and select one
  - Only visible if you have edit permission

---

SCENARIO 3: No Edit Permission

When you view a job order but DON'T have edit permission:

  Questionnaire: Full Time Position Questions
  Questions for full-time permanent positions

  Description:
  - Shows questionnaire name
  - Shows questionnaire description
  - No edit link (read-only view)

HOW TO USE IT
================================================================================

STEP 1: VIEW A JOB ORDER
  1. Go to Job Orders
  2. Click on any job order in the list
  3. Job order details page opens

STEP 2: LOOK FOR QUESTIONNAIRE SECTION
  Scroll down on the right side panel
  Find the row that says "Questionnaire:"

STEP 3: CHECK WHAT'S ASSIGNED
  ├─ If questionnaire is assigned:
  │  └─ Shows questionnaire name
  │
  └─ If no questionnaire:
     └─ Shows "No questionnaire assigned"

STEP 4: CHANGE QUESTIONNAIRE (If You Have Edit Permission)
  1. Click "Edit Questionnaire" or "Assign Questionnaire"
  2. You'll be taken to the Edit Job Order form
  3. Questionnaire dropdown is visible
  4. Select a different questionnaire if you want
  5. Click Save

TECHNICAL DETAILS

What Changed in Show.tpl:

Added after the "Type" row:

  <tr>
      <td class="vertical">Questionnaire:</td>
      <td class="data">
          <?php if (isset($this->questionnaireID) && $this->questionnaireID): ?>
              <strong><?php echo htmlspecialchars($this->questionnaireData['title']); ?></strong>
              <br />
              <small style="color: #666;">
                  <?php if ($this->getUserAccessLevel('joborders.edit') >= ACCESS_LEVEL_EDIT): ?>
                      <a href="<?php echo(CATSUtility::getIndexName()); ?>?m=joborders&amp;a=edit&amp;jobOrderID=<?php echo($this->jobOrderID); ?>#questionnaire">Edit Questionnaire</a>
                  <?php else: ?>
                      Assigned: <?php echo htmlspecialchars($this->questionnaireData['description']); ?>
                  <?php endif; ?>
              </small>
          <?php else: ?>
              <em style="color: #999;">No questionnaire assigned</em>
              <br />
              <small style="color: #666;">
                  <?php if ($this->getUserAccessLevel('joborders.edit') >= ACCESS_LEVEL_EDIT): ?>
                      <a href="<?php echo(CATSUtility::getIndexName()); ?>?m=joborders&amp;a=edit&amp;jobOrderID=<?php echo($this->jobOrderID); ?>#questionnaire">Assign Questionnaire</a>
                  <?php endif; ?>
              </small>
          <?php endif; ?>
      </td>
  </tr>

What Changed in JobOrdersUI.php:

Changed questionnaire loading to work for ALL jobs:

  // OLD CODE: Only loaded for public jobs
  if ($careerPortalEnabled && $data['public']) {

      $questionnaire = new Questionnaire($this->_siteID);
      $q = $questionnaire->get($data['questionnaireID']);
      if (is_array($q) && !empty($q)) { ... }
  }
