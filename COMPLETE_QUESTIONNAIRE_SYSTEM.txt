FEATURES IN DETAIL
════════════════════════════════════════════════════════════════════════════

FEATURE A: Auto-Selection in Forms
──────────────────────────────────────────────────────────────────────────

When Creating Job Order:
  1. Select Type: "FT" (Full Time)
  2. JavaScript detects change
  3. AJAX calls server: "Give me questionnaire for FT"
  4. Server returns: ID 5 = "Full Time Position Questions"
  5. Questionnaire dropdown auto-updates
  6. You see it selected instantly ✓

Benefit: No manual work, automatic & instant

---

FEATURE B: View Questionnaire in Job Order Details
──────────────────────────────────────────────────────────────────────────

When Viewing Job Order:
  1. Click on any job order
  2. Details page opens
  3. Right column shows "Questionnaire" row
  4. See which questionnaire is assigned
  5. Click "Edit Questionnaire" if needed

What You See:
  ├─ If assigned: Questionnaire name (bold)
  ├─ If assigned: Edit link (if you have permission)
  ├─ If empty: "No questionnaire assigned"
  └─ If empty: "Assign Questionnaire" link

Benefit: Transparency, quick access, no confusion

---

FEATURE C: Easy Editing from View Page
──────────────────────────────────────────────────────────────────────────

Want to Change Questionnaire:
  1. View job order details
  2. Click "Edit Questionnaire" link
  3. Edit form opens (pre-scrolled to questionnaire field)
  4. Change questionnaire if needed
  5. Save job order
  6. Back to view → Shows updated questionnaire ✓

Benefit: One-click access, no searching for field

---

FEATURE D: Career Portal Integration
──────────────────────────────────────────────────────────────────────────

Candidates Apply:
  1. See job on career portal
  2. Click "Apply"
  3. See job-type-specific questions
  4. Answer questions
  5. Submit application

Benefit: Better candidate filtering, organized questions

---

FEATURE E: Permission-Based Visibility
──────────────────────────────────────────────────────────────────────────

As Admin/Recruiter:
  └─ See full questionnaire info
  └─ See "Edit/Assign" links
  └─ Can change questionnaire

As Viewer (No Edit Permission):
  └─ See questionnaire name only
  └─ See description
  └─ No edit/change option

Benefit: Security, role-based access control
