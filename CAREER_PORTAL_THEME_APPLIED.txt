CUSTOMIZATION GUIDE:
════════════════════════════════════════════════════════════════════════════

To change colors, edit careersPage.css:

Find this section:
  :root {
      --primary: #2563eb;
      --primary-dark: #1d4ed8;
      ... etc
  }

Change values to your desired colors. All elements using --primary will
automatically update.

Example: To make buttons green instead of blue
  --primary: #10b981;
  --primary-dark: #059669;
  --primary-light: #d1fae5;
