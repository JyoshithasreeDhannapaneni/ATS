================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: Questionnaire not changing when I select type

DIAGNOSTICS:
1. Open browser Developer Tools (F12)
2. Go to Console tab
3. Select a job type
4. Check for errors

SOLUTIONS:
- Clear browser cache: Ctrl+Shift+Delete
- Hard refresh page: Ctrl+Shift+R
- Check if getQuestionnaireByJobType.php endpoint exists
- Verify questionnaire IDs in database:
  SELECT * FROM career_portal_questionnaire;
- Check AJAX request in Network tab (F12 → Network)

---

ISSUE: System shows "None" instead of questionnaire

SOLUTION:
1. Open /lib/JobOrderTypeQuestionnaires.php
2. Verify the IDs match the database
3. Run this SQL query:
   SELECT career_portal_questionnaire_id, title
   FROM career_portal_questionnaire;
4. Update IDs if they don't match
5. Clear browser cache
6. Test again

---

ISSUE: Questions don't appear for candidates

SOLUTION:
1. Login to ATS
2. Check job order details
3. Verify questionnaire_id field is populated
4. Go to Settings → Career Portal → Questionnaires
5. Verify the questionnaire has questions added
6. Add questions if needed
