HOW TO USE THE NEW FEATURE:
════════════════════════════════════════════════════════════════════════════

Scenario 1: View Questionnaire
  1. Click on any job order
  2. See details page
  3. Look at right column
  4. Find "Questionnaire" row
  5. See which questionnaire is assigned
  6. Done!

Scenario 2: Change Questionnaire
  1. Click on job order
  2. See "Questionnaire" row
  3. Click "Edit Questionnaire" link
  4. Edit form opens (pre-scrolled to questionnaire field)
  5. Change questionnaire if you want
  6. Click Save
  7. Back to view → Shows updated questionnaire

Scenario 3: Assign Questionnaire (if empty)
  1. Click on job order
  2. See "No questionnaire assigned"
  3. Click "Assign Questionnaire" link
  4. Edit form opens
  5. Select questionnaire from dropdown
  6. Click Save
  7. Back to view → Shows questionnaire assigned
