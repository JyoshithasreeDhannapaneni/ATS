HOW IT WORKS - SIMPLE EXAMPLE:

  Step 1: Recruiter creates job order
          Enters title, company, etc.
          Selects Type: "FT" (Full Time)
                    ↓
  Step 2: System auto-selects questionnaire
          JavaScript detects type change
          AJAX fetches correct questionnaire
          Dropdown updates instantly
                    ↓
  Step 3: Recruiter saves job order
          Job stored with questionnaire_id = 5
                    ↓
  Step 4: Candidate applies
          Career portal loads job with questionnaire ID 5
          Shows "Full Time Position Questions"
          Candidate sees FT-specific questions
                    ↓
  Result: Better candidate screening! ✓

IF SOMETHING DOESN'T WORK:

  Problem: Questionnaire not auto-selecting

  Solution:
    1. Press F12 in browser (Developer Tools)
    2. Go to Console tab
    3. Select a type in dropdown
    4. Check for red error messages
    5. If errors, report them
    6. If no errors, clear cache (Ctrl+Shift+Delete)
    7. Hard refresh (Ctrl+Shift+R)
    8. Try again

  Problem: Shows "None" instead of questionnaire

  Solution:
    1. Verify IDs in /lib/JobOrderTypeQuestionnaires.php
    2. Check database has those questionnaires:
       SELECT * FROM career_portal_questionnaire;
    3. Make sure IDs match (should be 2, 3, 4, 5, 6)
    4. Update file if IDs don't match
    5. Clear cache and try again
